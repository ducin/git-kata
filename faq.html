Do I need to know English to attend?

Although the website is in english, most of the trainings will be in Polish. Only is there are any foreigners interested mentors may switch to English on demand.

I don't know how to register

We have provided a geeky way of registration, but if that's too problematic for you - you should visit the BitBucket pull request help page. If that doesn't help then you should keep in mind that you may have problem with understanding the workshops as well (it isn't an introduction to Git, we're aiming current, but not yet powerful users). Start with Git tutorial and get more familiar with Git. There is still time.

I'm not a Git user but want to see the event. May I come?

If there is any other reason than improving your Git - you're welcomed. Just please contact us first.
